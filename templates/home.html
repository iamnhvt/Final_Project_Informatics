HOW WE COLLECT THEM?

We do recognize that HIV is the century disease that ruins our world considerably over that past 20 years. After finishing examining many set of data, it seems feasible to us that HIV incidence in one country might correlate to its development in economics .

Finally, we ended up selecting 5 distinct set of datas, each set is broken up in countries:
  1. Males HIV Incidence (15-24)
  2. Females HIV Incidence (15-24)
  3. Total GDP and GDP Per Capita
  4. Total Population
  5. Income and Region
Data Relationship Website Features
DATA PROCESSING
Manipulating And Processing Efficiently 6 Sets Of Data With More Than 20000 Cells.
Pre-Processing
Google spreadsheet is mainly used to pre-process the raw csv data, which involves deleting some errors cell, and clean the empty rows. We also manually change some cells due to semantic errors.
Strict Data Analysis Process
We strictly follow the 5 steps in data analysis process, which includes: Sourcing, Understanding, Cleaning, Transformation, and Feature Selection.
Efficient Data Storage
All the data in this project is stored and managed by just 2 dictionary data structure of Python. Interestingly, these two data structures are used together to find relationship easily and accurately.
DATA FILTERING
Extracting The Key Data And Presenting In A General Table. Besides, Users Can Select The Information They Want By Pivot Table Form.
Error Handling
We create a unique function to handle all possible errors related to input, value, and aggregation, then forward to a new page to show the error description.
Efficient Aggregation And Filter Algorithm
Flexible and wide-range filter value and Aggregation options is provided in this web platform. All filtered information is processed and seleted accurately at server side (using Python).
Flexible Pivot Table
We use the same table of general information. All the filtered data is passed to the table cell by using Flask framework. Therefore, the table is very responsive, and has many features for users.
Visual Complementation
Every pivot table is visualized by a bar chart. The bar chart is created dynamically by JavaScript, which collects all the data in table cells and make the chart based on Google Chart API. Besides, each cell is colored based on it values.
DATA VISUALIZATION
Providing Visual Complementation For Almost Every Set And Relationship Of Data.
Google API and JavaScript
Google charts API is the unique outsourced API we use to display visualization in this project. Besides, we also use JavaScript to make the chart more responsive and user-friendly.
Wide Variety Of Charts
By Investigating in many possible relationship of data, we can find a wide number of types to represent it. Some common forms are Line chart, Bar chart, Pie chart, Geo chart, and Bubble chart.
Two Ultimate Charts
We did provide the two ultimate charts, which represent the data in general of many countries in the period from 1990 to 2015. This gives a better understanding of problems, and more accurate hypothesis.
Detailed Charts
Beside the ultimate charts, The chart related to the relationship of many aspects of HIV is provided to maximize the understanding of problems.
